12-15 英语考试题库

单选题

  1. Which of the folllowing are not the advantage of the cloud（B）
    A. Reliable B.Steady C. Adaptable D.Multi - tenancy

    译： 以下哪项不是云的优势（B）

    A.可靠的B.稳定的C.适应性强的D.多租户

  2. Bigdata has the following 4vs characteristic except （B）
    A. Volume B.Victory C. Variety D.Veracity
    译： Bigdata具有以下4vs特性，除了（B）
    A、体积 B.卷胜利C.变种D.真实性

  3. The EAS often used in ( A)
    A. Supermarket B. Hospital C. Police Office D. Home
    译：（A）中常用的EAS
    A、 超市B医院C警察局D家

  4. Zigbee is a low-power, ( D ) and close proximity wireless ad hoc network
    A. big scale B.long distance C.high data rate D.low data rate
    译： Zigbee是一种低功耗、近距离无线adhoc网络
    A、 大比例尺B.长距离C.高数据速率D.低数据速率

  5. Antivirus Software is computer software used to prevent, detect and (A) malicious software.
    A. remove B.download C.update D.check
    译： 防病毒软件是用来预防、检测和（A）恶意软件的计算机软件。
    A、 删除B.下载C.更新D.检查

  6. Which of the following are not belongs to topology( D)
    A. Star B.Ring C.Bus D.Squre
    译： 以下哪项不属于拓扑（D）
    A、 星B环C巴士D广场

  7. The relationship of the data model except ( D)
    A. 1:M B. 1:1 C .M:M D.P:P
    译： 除（D）外的数据模型的关系
    A、 1:M B.1:1 C.M:M D.P:P

8.Which of the following are not High-level language( C)
A.Java B.Java Script C.Assembly language D.Android
译： .下列哪项不是高级语言（C）
A、 javab.Java脚本C.汇编语言D.Android

9.Which of the following are not the components of the system software ( B)
A.Operating system B.Physical hardware
C.Device drivers D.Utilities
译： 以下哪项不是系统软件（B）的组件
A、 操作系统B.物理硬件
C、 设备驱动程序D.实用程序

10.Which of the following device are not belongs to the input device( A)
A.printer B.joysticks C.webcams D.microphones
译： 下列哪个设备不属于输入设备（A）
A、 打印机B.操纵杆C.网络摄像头D.麦克风


# 判断题
1. Machine learning is a type of artificial intelligence (AI)（Y） 译：机器学习是一种人工智能（AI）（Y）
2. Direct attached storage is computer storage that is connected to one computer and accessible to other computers.（N） 译：直连存储是连接到一台计算机并可供其他计算机访问的计算机存储器(N)
3. Patch is a piece of hardware designed to fix or improve a computer program（N） 译：补丁是一种硬件，用于修复或改进计算机程序（N）
4. Recovery is a process of salvaging inaccessible data. （Y） 译： 恢复是恢复无法访问的数据的过程。（Y）
5. Virus,Worm and Trojan Horse are nearly the same.（N） 译： 病毒、蠕虫和木马几乎是一样的(N)
6. Host is any end-user computer system that connects to a network.（Y） 译：主机是任何连接到网络的最终用户计算机系统(Y)
7. Browser is an application used to access the World Wide Web.（Y） 译： 浏览器是用于访问万维网的应用程序(Y)
8. The DBMS manages the interaction between the end user and the database.（Y） 译：数据库管理系统管理最终用户和数据库之间的交互(Y)
9. Computer language is programming language.（N） 译：计算机语言是编程语言(N)
10. There are no difference among virus ,worms and Trojan horse.（N） 译： 病毒、蠕虫和木马没有区别(N)
11. Application software is one designed to help the user to perform specific tasks.（Y） 译：应用软件是设计用来帮助用户执行特定任务的软件(Y)
12. Although computer hardware and software require each other ,the hardware can be used on its own.（N） 译： 虽然计算机硬件和软件相互需要，但硬件可以单独使用(Y)
13. Linux can be modified and distributed by anyone around the world.（Y） 译： Linux可以被世界各地的任何人修改和分发(Y)
14. Windows comes preloaded on most new PCs, which helps to make it the most popular operating system in the world.（N） 译：大多数新电脑都预装了Windows，这有助于使它成为世界上最受欢迎的操作系统（N0）
15. The bandwidth and the broadband are the same thing（N） 译：带宽和宽带是一样的（N）
16. An operating system is the most important software that runs on a computer.（Y） 译： 操作系统是在计算机上运行的最重要的软件(Y)
17. The term network and Internet are the same thing（N） 译： 网络和因特网是一回事（N）
18. Mouse and keyboard are output device （N） 译： 鼠标和键盘是输出设备（N）
19. Read-Only Memory is a kind of temporary storage.（N） 译： 只读存储器是一种临时存储器(N)
20. Computer hardware consists of system unit and peripheral device.（Y） 译：计算机硬件包括系统单元和外围设备(Y)
