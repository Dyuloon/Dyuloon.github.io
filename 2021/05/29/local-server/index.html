将自己的电脑当作web服务器运行jsp

发表于 2021-05-29 更新于 2024-12-18 分类于 javaweb ， 服务器 阅读次数： Valine：

服务器本质是一台24小时开机的个人电脑
所以自己的电脑也可以当作服务器使用
以下是操作方法
操作系统：win10

注意

不用的时候记得禁用防火墙入站规则
小心被黑掉（以防万一，虽然黑客没那么闲）

设置防火墙入站规则

打开 windows 防火墙，选择进入高级设置，点击入站规则，新建规则

规则类型

端口



端口号

你的服务器程序端口号是什么就写什么：8888



操作

默认



配置文件

全选



名称

没什么好说的



服务器配置 - tomcat 为例

可以将服务器直接配置到 eclipse 开启服务器即可
也可以按流程配置到 tomcat


网站浏览

http:/你的电脑ip地址:tomcat服务器端口号/项目名/页面
如：
http://127.0.0.1:8888/JspProject/index.jsp

jsp 服务器
JSP — 九大内置对象
JSP - request - get 和 post
